# 🙏 기도노트 (Prayer Note App)

간단하고 아름다운 기도 관리 웹 애플리케이션

## ✨ 기능

- 📝 간단한 기도 작성 및 관리
- 🏷️ ACTS 기도 분류 (간구, 감사, 찬양, 자백)
- 📊 기도 상태 추적 (W: 기다림, Y: 응답됨, N: 응답안됨)
- 💾 로컬 스토리지 자동 저장
- 📱 반응형 디자인 (모바일 지원)

## 🚀 시작하기

### 설치 없이 바로 실행
1. `index.html` 파일을 브라우저에서 열기

```bash
# 의존성 설치

# 개발 서버 실행
